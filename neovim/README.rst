====================
neovim configration
====================


config file location
~~~~~~~~~~~~~~~~~~~~~~

- PATH of ``init.vim`` and ``dein``

.. code:: bash

   tree ~/.config -a -P "*vim*" --prune
   # ~/.config
   # └── nvim
   #     └── init.vim

   tree ~/.config -a -P "*dein*" --prune
   # ~/.config               
   #  └── nvim              
   #      ├── dein.toml     
   #      └── dein_lazy.toml
